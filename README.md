# 📝 Blogging Platform

A full-featured Blogging Platform built with the **MERN stack** (MongoDB, Express, React, Node.js). Supports full blog publishing, rich text editing, user authentication, profiles, admin dashboard, and more.

---

## 🚀 Features

- ✍️ **Rich Text Editor** with [Quill.js]
- 🔐 **JWT Authentication**
- 👥 **User Profiles** with blogs and info
- 📰 **Single Blog View**
- 💬 **Comments and Likes**
- 📁 **Category Filtering**
- 🔎 **Search Functionality**
- 📊 **Admin Dashboard**
- 🌐 **Responsive UI**

---

## 🛠 Tech Stack

### 📦 Frontend
- React.js (with Hooks)
- Tailwind CSS
- React Router DOM
- Redux Toolkit (optional)
- Axios
- Quill.js
- date-fns

### 🔧 Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JSON Web Tokens (JWT)
- Multer (for file uploads)
- bcrypt (for password hashing)
- dotenv
